# Ecommerce Sales Intelligence Dashboard

This repository contains an end-to-end ecommerce sales analysis project featuring:

- SQL scripts for database setup and business analysis
- Dataset for ecommerce sales transactions
- Power BI dashboard template and report file
- Documentation for project notes and reporting insights

## Project Structure

- Dataset/ - Sales data files
- SQL/ - SQL queries for setup, validation, and analysis
- Power BI/ - Dashboard files
- Documentation/ - Project notes and report files

## Technologies Used

- SQL
- Power BI
- Excel/CSV data

## Purpose

This project is designed to analyze ecommerce sales performance, profitability, customer behavior, and shipping efficiency.
